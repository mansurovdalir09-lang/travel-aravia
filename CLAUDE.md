# Travel Aravia — Project Context

## Проект
- Название: Travel Aravia (Трэвел Аравия)
- Описание: Многостраничный тревел-сайт по Саудовской Аравии для русскоговорящих
- Стек: HTML/CSS/JS (без фреймворков)
- Хостинг: GitHub Pages

## Бренд
- Primary: #5C2D1A
- Accent: #C4875A
- Sand: #E8D5B7
- Dark: #1A0F0A
- White: #FFFFFF
- Шрифты: Cormorant Garant (заголовки), Inter (текст)
- Логотип: assets/images/logo.jpg

## Структура страниц
- index.html — Главная
- tours.html — Каталог туров

- events.html — Мероприятия Эр-Рияда
- hotels.html — Отели
